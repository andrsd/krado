Getting started
===============

Prerequisites
-------------

- CMake
- C++17 compiler
- fmt
- exodusIIcpp
- OpenCASCADE
- Eigen3
- Boost
- Python 3.6 or later
- pybind11
- Doxygen (optional)

Build from source
-----------------

.. code-block:: bash

    mkdir build
    cd build
    cmake ..
    make
